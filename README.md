# 🚀 AptiCode

🌐 Live Demo: https://apti-code.vercel.app/

---

## 📌 About the Project

**AptiCode** is a full-stack web application designed to help students practice **aptitude problems** in a structured and interactive way.

It provides topic-wise problem solving, progress tracking, and a clean modern interface — inspired by popular coding platforms, but focused on aptitude preparation.

---

## ✨ Features

* 📚 Topic-wise Aptitude Problems
* 🔍 Search and Filter Problems
* 📊 User Dashboard & Progress Tracking
* 🔐 Authentication (Login / Signup)
* 🎯 Difficulty Levels & Status Tracking
* 🌙 Clean Dark-Themed UI

---

## 🛠️ Tech Stack

### Frontend

* React.js
* Bootstrap / CSS

### Backend

* Node.js
* Express.js

### Database

* MongoDB Atlas

### Deployment

* Frontend → Vercel
* Backend → Render

---

## ⚙️ How It Works

1. User logs in / signs up
2. Selects a topic (Arithmetic, Time & Work, etc.)
3. Solves problems
4. Progress is tracked and stored

---

## 🚀 Setup Instructions

### Clone the repository

```bash
git clone https://github.com/sricharan-213/AptiCode.git
cd AptiCode
```

### Backend Setup

```bash
cd backend
npm install
npm start
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

---

## 🔐 Environment Variables

Create a `.env` file in backend:

```env
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
PORT=5000
```
